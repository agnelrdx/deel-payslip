# Deel Payslip

Capacitor app to view your payslip.

Following commads are available.

## Install dependencies

```
yarn
```

## Run webapp

```
yarn dev
```

## Run android using android studio

```
yarn build:android
```

## Run ios using xcode

```
yarn build:ios
```
